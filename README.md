# PDF Converter Frontend

A modern web application for converting PDF files to text and images.

## Overview

This is a Next.js-based frontend application that provides PDF conversion tools including:
- PDF to Text conversion
- PDF to Image conversion

## Features

- Modern, responsive UI built with Next.js and React
- Dark/Light theme support
- Clean and intuitive user interface
- Multiple PDF conversion tools

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.
